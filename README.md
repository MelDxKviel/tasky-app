# Tasky App

Donwload using git

```sh
$ git clone https://github.com/MelDxKviel/tasky-app.git
```

Configure .env

```sh
$ cp ./backend/example.env ./backend/.env
```

Start with build

```sh
$ docker-compose up --build
```

Access project

[http://localhost](http://localhost)


API Docs

[http://localhost/api/docs](http://localhost/api/docs)
